# Example DUP Project setup

This is an example DUP project which should not be used in production, but is meant to showcase how a DUP pipeline has to be configured.

To run through the project steps follow the following steps:

1. Set Up your data node and load testdata into your fhir server
2. Execute the `set-example-dimp-and-restart-pseudonymizer.sh`
3. Execute the `example-project-create-namespaces-vfps.sh`
4. Execute the `example-project-aether-exec.sh`
5. check your output in jobs
